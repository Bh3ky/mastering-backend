# Backend Development with Python Roadmap

What to master??

1. Language knowledge 

    - [ ] Object-oriented programming
    - [ ] Metaclasses
    - [ ] Decorators
    - [ ] Generators
    - [ ] Context managers
    - [ ] Iterators
    - [ ] Asynchronous programming
    - [ ] Global interpreter
    - [ ] Multiprocessing
    - [ ] Multithreading
    - [ ] Modules
    - [ ] Libraries
    - [ ] Python conventions

2. Data structures and algorithms

    - [ ] Arrays
    - [ ] Linked lists
    - [ ] Heaps
    - [ ] Binary trees
    - [ ] Time complexity
    - [ ] Runtime performance of the code

3. Basic web development 

    - [ ] Basic HTML, CSS & JavaScript
    - [ ] Frontend frameworks like React, Angular, and Vue


4. API Development

    - [ ] REST APIs
    - [ ] HTTP
    - [ ] Different HTTP methods
    - [ ] Python frameworks like Django, FastAPI, and Flask
    - [ ] Authentication and authorisation:
        * What tokens are?
        * What JWT tokens are?
        * What OAuth is?
        * How to secure an API?
        * How rate limiting works?
    - [ ] Module: Pydantic


5. Databases

    - [ ] Types of databases:
        1. SQL [understand how to create tables, insert values, and run queries. Be comfortable with more advanced queries]
        2. NoSQL [NB: understanding how document-style data works including nested objects and flexible structures. Checkout MongoDB or Firebase]
    - [ ] Concepts: Inserting data and retrieving data


6. ORMs [Object Relational Mapping]
    * Object relational mapping (ORM) - used to map data to a Python object, so that we can use Python syntax to make changes to the database or query data without writing custom SQL queries every time. 
        * SQLAlchemy - let’s write Python code that changes or queries data in the database.

    - [ ] ORMs
    - [ ] Basic database design
    - [ ] Writing safe queries
    - [ ] Preventing SQL injection
    - [ ] Handling data properly
    - [ ] Checkout GraphQL

7. Linux and the Command Line

    - [ ] Know how to navigate the command line
        - [ ] Create files
        - [ ] Remove files
        - [ ] Modify configuration files
        - [ ] SSH into a remote server or virtual private server 
        - [ ] Basic shell scripts

8. Tooling

    - [ ] Git and version control
    - [ ] Testing
        - [ ] Unit tests
        - [ ] Integration tests
        - [ ] System tests
        - [ ] How to test APIs built with frameworks like Flask or Django

9. Debugging

    - [ ] Using the IDE for debugging


10. API Testing

    - [ ] Familiarity with tools like curl or Postman


11. Deployment

    - [ ] Docker and Kubernetes
    - [ ] CI/CD
    - [ ] Automated testing
    - [ ] Automated deployments
    - [ ] GitHub Actions
    - [ ] Cloud platforms such as Azure, Google Cloud, or AWS
